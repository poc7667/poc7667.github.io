angular-cheatsheet

Show un-safe string as HTML

Add a filter

angular
    .module('altairApp').filter('unsafe', function($sce) {
    return function(val) {
        return $sce.trustAsHtml(val);
    };
});

In the view

<p ng-bind-html="product.description | unsafe"></p>
